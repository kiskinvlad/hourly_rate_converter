# HourlyRate converter

## Simple app created using PyQt5 for osX

## How to build easy

```
pyinstaller main.py --noconsole --ico=hri.icns --onefile --name=HourlyRateHelper

```

## How to create .dmg
```
appdmg spec.json hourly_rate_helper.dmg

```
